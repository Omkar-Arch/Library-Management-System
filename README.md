# 📚 Library Management System (C++)

A console-based Library Management System developed in **C++** using **Object-Oriented Programming (OOP)** principles. The project demonstrates CRUD operations, modular project organization, and the use of the C++ Standard Template Library (STL).

---

## ✨ Features

- 📖 Add a new book
- 👤 Register a new user
- 🔍 Search books by title
- 📋 List all books
- 📚 Assign books to users
- ❌ Remove books using ISBN
- 🏗️ Modular project structure using separate header and source files

---

## 🚀 How to Compile

Using g++:

```bash
g++ src/main.cpp src/Book.cpp src/User.cpp src/Library.cpp -I include -o LibraryManagement
```

Run:

```bash
./LibraryManagement
```

Windows:

```bash
LibraryManagement.exe
```
